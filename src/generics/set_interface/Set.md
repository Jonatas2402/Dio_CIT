A interface set é uma coleção que não pode conter elementos duplicados.
Essa interface representa o conceito matemático de um conjunto e é usada para
representar um conjuntos, como um baralho de cartas.
A plataforma Java possui três implementações de set de uso geral: HashSet, TresSet, LinkedHashSet.
A interface set não permite o acesso aleatório a um elemento na coleção.
Para percorrer os elementos do set você pode usar um iteretor ou um loop foreach.
